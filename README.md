 My First Python Bot (The Struggle is Real)
Привет! Это мой самый первый проект на Python. Я учусь создавать Telegram-ботов с помощью библиотеки aiogram

 Что я прошел, пока делал это:
Понял, что ООП — это не страшно (почти).

Узнал, что асинхронность (async/await) заставляет мозг кипеть.

Сражался с блокировками, прокси и VPN 

Научился прятать токены в .env, чтобы меня не взломали

🛠 Что умеет этот бот:
Здороваться: Отвечает на команду /start очень дружелюбным сообщением.

Работать эхом: Повторяет всё, что ты ему напишешь. Простой, но гордый механизм!

🚀 Как запустить (для тех, у кого есть VPN):
Клонируй репозиторий: git clone <ссылка>

Установи зависимости: pip install aiogram python-dotenv

Создай файл .env и положи туда свой BOT_TOKEN.

Запускай: python main.py
